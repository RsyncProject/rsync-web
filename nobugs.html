rsync bug tracking

The Jitterbug database is no longer active.

If you think you have found a bug in rsync we would appreciate you checking the list of current issues and the TODO file in the source distribution to see if it's already known. Please send bug reports or suggestions to the rsync mailing list. You can send patches to the list too.

Before you send a bug report, please take a couple of minutes to read Simon Tatham's advice on how to report bugs.
