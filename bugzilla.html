rsync bug tracking

If you think you have found a bug in rsync we would appreciate you checking the list of current issues and the bugzilla database to see if it's already known.

Also, please take a couple of minutes to read Simon Tatham's advice on how to report bugs.

To report a bug or make suggestions, use one of these methods:

  • Use this bugzilla form (this does require that you have created a bugzilla account, but that is a very simple process).
  • Send an email to the rsync mailing list, which is at rsync@lists.samba.org. You can send patches to the list too.

Thanks for helping out!
