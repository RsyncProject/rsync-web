rsync download

You can download rsync via ftp, http, cvs or rsync.

http

  • The source code is available here
  • Binaries for a number of platforms are available here

ftp

  • The source code is available here
  • Binaries for a number of platforms are available here

rsync

  • The rsync ftp site is available via anonymous rsync at rsync://rsync.samba.org/rsyncftp/
  • An unpacked copy of the latest development version can be obtained from rsync://rsync.samba.org/ftp/unpacked/rsync/

cvs

  • Details on how to access the anonymous cvs archives on cvs.samba.org are available here
  • Browse cvsweb here.

Contributing binaries

If you can build binaries for a platform that we currently don't offer binaries for then please let me know (via the bug tracking system). I'll then give you a way of uploading binaries to rsync.samba.org. Please only do this if you have the time to build and package binaries in a timely fashion for every rsync release.
